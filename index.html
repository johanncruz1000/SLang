SLang

Compiled Programming Language with Native C++ Power

SLang is a compiled programming language designed to combine clean syntax, high predictability, and native C++ performance — without boilerplate.

  • No int main()
  • Optional semicolons
  • No wrappers

Why SLang?

Most languages force you to choose:

  • simplicity or performance
  • scripting or system-level power
  • safety or control

SLang refuses that trade-off.

Basic Features

  • Compiled language
  • Native C++ code injection
  • Predictable behavior (explicit types & blocks)
  • No int main() required
  • Optional semicolons
  • Import system
  • Script-like syntax with system-level power

Philosophy

SLang trusts the programmer.

It does not try to protect you from yourself. It gives you full control — and expects you to know what you're doing.

If you want safety rails, this language is not for you.

Functions & Loops

func int fib(int n){
    if(n <= 1){
        return n
    }else{
        return fib(n-1) + fib(n-2)
    }
}

print(String(fib(10)))

func void count(){
    for(var i=0; i<10;i++){
        print("numero atual:" + String(i))
        print("\n")
    }
}
